# 1.认识数据类型

## 1.数据类型分类

```python
数据类型可分为：数值，布尔(bool)，字符串(str)，列表(list)，元组tuple，集合(set)，字典(dic)
其中数值分为"整型(int),浮点型(float)“
布尔型分为：真(True),假(false)

```
